# textil — 操作ガイド（Phase 9 時点）

## 起動方法

```bash
# リポジトリルートから
bun run dev

# または web パッケージ単体で
cd packages/web
bun run dev
```

ブラウザで `http://localhost:3000/studio` を開く。

## テキスト生成（Text モード）

サイドバー上部の **[Text]** タブを選択。

| 操作 | 説明 |
|------|------|
| テキスト入力欄 | ASCII アートに変換するテキストを入力（ページ読み込み時に自動フォーカス） |
| フォントピッカー | 12 種類のフォントからサムネイルで選択 |
| Width スライダー | 出力幅を 40〜160 文字で調整（ステップ 4） |

入力内容は **150ms デバウンス**でリアルタイムにプレビューへ反映される。

---

## 画像変換（Image モード）

サイドバー上部の **[Image]** タブを選択。

### 画像のアップロード

- メインエリアのドロップゾーンに **PNG / JPEG をドラッグ&ドロップ**
- またはドロップゾーンをクリックしてファイルダイアログで選択

### 調整パラメータ

| パラメータ | 範囲 | 説明 |
|-----------|------|------|
| Width | 40〜160 | 出力の横幅（文字数） |
| Charset | standard / braille / block | 文字セット切り替え |
| Contrast | −1.0〜1.0 | 明暗の強調度 |
| Threshold | 0.0〜1.0 | 明暗の境界値 |

パラメータ変更は **200ms デバウンス**でプレビューへ反映される。

## グリッドエディタ

プレビューが生成された状態でサイドバーの **[Edit grid]** をクリックするとエディタモードへ切り替わる。

### ツールバー

| ツール | キー | 説明 |
|--------|------|------|
| Pencil（鉛筆） | `P` | クリック / ドラッグで任意の文字を書き込む |
| Eraser（消しゴム） | `E` | クリック / ドラッグでセルをスペースに戻す |
| Select（選択） | `S` | ドラッグで矩形選択、移動・削除が可能 |
| Text（テキスト） | `T` | クリックでカーソルを配置し、キーボードで直接入力 |

> ツールキー（P / E / S / T）はテキストモード中は無効。

### 鉛筆ツール

- ツールバーの文字入力欄（1 文字）で描画文字を変更できる
- ドラッグ中はポインタが通過したすべてのセルに描画される

### 選択ツール

| 操作 | 動作 |
|------|------|
| ドラッグ | 矩形を選択 |
| 選択範囲内でドラッグ | 選択範囲を移動 |
| `Delete` / `Backspace` | 選択範囲をスペースでクリア |

### テキストツール

| キー | 動作 |
|------|------|
| 文字キー | カーソル位置に文字を入力し右へ進む |
| `Backspace` | 1 文字前に戻って削除 |
| `Enter` | 次の行の先頭へ移動 |
| `Tab` | 4 列右へ進む |
| `←` `→` `↑` `↓` | カーソル移動 |
| `Escape` | テキストモードを終了 |

### Undo / Redo

| 操作 | キー |
|------|------|
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Y` |

最大 100 ステップの履歴を保持。

### Find & Replace

ツールバー右側のパネルに検索文字（1 文字）と置換文字（1 文字）を入力して **[Replace All]** を押す。置換された件数がツールチップで表示される。

### ズーム

ツールバーのスライダーで **50%〜400%**（25% ステップ）に拡大縮小できる。

## エクスポート

サイドバーの **[Export]** をクリックするとモーダルが開く。

### 対応ターゲット

| ターゲット | 出力形式 | 操作 |
|-----------|---------|------|
| **Plain Text** | プレーンテキスト | クリップボードにコピー |
| **GitHub** | Markdown コードブロック（` ```\n...\n``` `） | クリップボードにコピー |
| **Terminal** | ANSI カラーコード付きテキスト | クリップボードにコピー |
| **Discord / Slack** | プレーンテキスト（2000 文字制限チェック付き） | クリップボードにコピー |
| **PNG** | 画像ファイル（monospace フォントで描画） | `ascii-art.png` としてダウンロード |
| **React Component** | `<pre>` ラップの TypeScript コンポーネント | `AsciiArt.tsx` としてダウンロード |

### 注意事項

- **Discord / Slack**: グリッドが 2000 文字を超える場合は警告が表示される
- **GitHub**: 幅が 80 列を超える場合は Markdown プレビューで折り返しが起きる可能性がある
- **PNG**: ブラウザの Canvas API を使用してレンダリング（背景 `#18181b`、文字 `#d4d4d8`、14px Courier New）
- クリップボード API は **HTTPS 環境**が必要。ローカル開発（`localhost`）では問題なく動作する

### エクスポートモーダルを閉じる

モーダル外をクリック、または `Escape` キーで閉じる。

## キーボードショートカット一覧

| キー | 効果 |
|------|------|
| `P` | 鉛筆ツールに切り替え（エディタ内、非テキストモード） |
| `E` | 消しゴムツールに切り替え |
| `S` | 選択ツールに切り替え |
| `T` | テキストツールに切り替え |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Delete` / `Backspace` | 選択範囲を削除（選択ツール使用中） |
| `Escape` | テキスト入力モードを終了 |
